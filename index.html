Event Planner

This repository contains a Flask application for event planning and booking.

GitHub Pages can only show static files. This root-level page is here so the site does not display the repository README.

Important: The app itself requires Python and Flask to run, so it cannot be hosted on GitHub Pages as a dynamic web application.

To run the app locally, use:

python app.py

Or deploy the repository to a Python/Flask hosting service such as Render, Railway, PythonAnywhere, or Heroku.
